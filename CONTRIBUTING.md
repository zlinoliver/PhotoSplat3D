# Contributing

Thanks for contributing to PhotoSplat3D / 图生高斯3D.

## Development Setup

1. Create a Python 3.13 environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run checks before opening a PR:

```bash
ruff check src
ruff format src
```

## Pull Request Guidelines

1. Keep PRs focused and small.
2. Add/update docs when behavior changes.
3. Include reproduction and verification steps in the PR description.
4. Avoid unrelated refactors in feature/fix PRs.

## Reporting Issues

Please include:

1. OS + Python version
2. How to reproduce
3. Error logs (if available)
4. Expected vs actual behavior

## Security

Please do not report security issues publicly. See `SECURITY.md`.
